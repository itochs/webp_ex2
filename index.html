HTML要素

ul要素およびli要素

説明

ul要素はリスト表示，つまり箇条書きをする際に使う要素である．li要素は，その箇条書きの項目を表すタグである

記述例


            <ul>
                <li>one item</li>
                <li>second item</li>
            </ul>
        

            <ul>
                <li>one item</li>
                <li>two item</li>
                <ul>
                    <li>second-one item</li>
                    <li>second-two item</li>
                </ul>
            </ul>

表示

  • one item
  • two item
  • one item
  • two item
    • second-one item
    • second-two item

a要素

説明

a要素は，他のページへのリンク，またはページ内の場所へのリンクを埋め込むためのタグである．

記述例


            <a href="https://www.google.com">Google browser</a>

            <p>文章の途中で<a href="https://www.google.com">リンク</a>を入れる</p>

表示

Google browser

文章の途中でリンクを入れる

nav要素

説明

nav要素は，他のページへのリンク，またはページ内の場所へのリンクをまとめたセクション

記述例


            <nav>
                <ul>
                    <li>other page</li>
                    <li>in page</li>
                </ul>
            </nav>

表示

  • other page: Google browser
  • in page
